# predection-model

> Modèle de prédiction Python.

## Getting Started

### Prerequisites

- Python 3.8+
- [uv](https://github.com/astral-sh/uv)

### Installation

```bash
git clone <repo-url>
cd predection-model
uv sync
```

### Usage

```bash
uv run python main.py
```
