# predection-model

> Modèle de prédiction Python.

## Getting Started

### Prerequisites

- Python 3.12+
- [uv](https://github.com/astral-sh/uv)

### Installation

```bash
git clone https://github.com/nirre55/predection-model.git
cd predection-model
uv sync
```

### Usage

```bash
uv run python main.py
```
